# DESCRIPTION #
Ce projet est un shell basé sur un noyau Linux. Il répond aux principales fonctions que l'on peut retrouver dans un shell normal. Nous pouvons, par exemple, être capable de définir ou modifier des variables d’environnement. Le shell peut également exécuter diverses commandes, qu’elles soient internes ou externes, et les rediriger, en entrée ou en sortie. 

# SYSTEME REQUIS #
Système Linux ou MacOS

# LANCEMENT DU SHELL #
Dans le dossier SE_IMR_PIGNET_RANNOU faire :
./bin/shell

# LANCEMENT DU SHELL AVEC LE SCRIPT DE SCENARIO DE TEST #
Dans le dossier SE_IMR_PIGNET_RANNOU faire :
./bin/shell script.imr
